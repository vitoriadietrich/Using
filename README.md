# Neumorphism Keyboard Design using only HTML & CSS

Um projeto simples de **teclado com efeito neumorphism** usando apenas HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **keyboard com estilo neumorphism**, utilizando sombras suaves para um visual moderno e elegante.

## 🛠️ Tecnologias

- HTML
- CSS (box-shadow, neumorphism)

## 🎯 Funcionalidades

- Layout de teclado
- Efeito neumorphism
- Design moderno e minimalista

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
